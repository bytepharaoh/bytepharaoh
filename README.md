### 📂 Featured Projects

**[Mobix](https://github.com/bytepharaoh/Mobix)** — Ride-sharing microservices platform
4 independently deployable services (gateway, trip, driver, payment) communicating via gRPC, RabbitMQ, and WebSockets. Redis caching cut average DB read latency by ~65%; full distributed tracing with OpenTelemetry + Jaeger.

**[Bar108](https://github.com/bytepharaoh/Bar108)** — Live production restaurant API
30+ REST endpoints with strict 3-layer architecture (handler/service/repository). SQLC for type-safe queries, JWT + bcrypt auth, Goose migrations, 13MB Docker image, table-driven tests with GoMock and race detector.

**[BankSystem](https://github.com/bytepharaoh/BankSystem)** — Banking backend
Deadlock-safe money transfers via consistent account ordering in PostgreSQL transactions. JWT auth with ownership-based authorization and full test coverage across layers.
